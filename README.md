# Lightweight-Skin-Lesion-Diagnosis
Lightweight few-shot learning framework for skin lesion classification
# Requirements
Install dependencies:
pip install -r requirements.txt
# Usage
1.  Prepare datasets (HAM10000 / sd198) using the provided split files.
2.  Train the model:
    python train_generalization.py
3. test the model:
    python sd198_expert_test.py
